Each week, you will be asked to complete a short task based on the topic from class. An example of how to portfolio can be found in the `Sample portfolio` directory.

## Unit Structure

### Weekly topics:

- **Week 01:** Pixels and Colour
- **Week 02:** Manipulating and Analyzing Pixels
- **Week 03:** Vector Graphics
- **Week 04:** Filtering Images
- **Week 05:** Convolution and Dithering
- **Week 06:** Synthesizing Sound
- **Week 07:** Motion, Vectors, Forces
- **Week 08:** Particles, Systems
- **Week 09:** Simulation & Behaviour
- **Week 10:** Final Portfolio Submission
